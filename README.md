#FilterBased-Ecommerce

A simple e-commerce app built with React that allows filtering products by category, price, color.

#Features

- Product listing with filters
- Search functionality
- Wishlist (love items)
- Star rating display

#Installation

1. Clone the repo: `git clone https://github.com/yourusername/FilterBased-Ecommerce.git`
2. Install dependencies: `npm install`
3. Start the app: `npm run dev`
